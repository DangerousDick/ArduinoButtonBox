* When setting up the sketch ArduinoJoystickLibrary.zip needs to be installed.
  * Arduino IDE -> Sketch -> Include Library -> Add .zip Library.
* Use TinyCad online application (https://online.tinycad.net/fs/) to read/edit .dsn files.
* Use DIY Layout Creator (https://github.com/bancika/diy-layout-creator) to open .diy files.
